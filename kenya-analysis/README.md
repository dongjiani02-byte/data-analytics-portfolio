Kenya Economic Data Analysis
Project Overview
This project analyzes Kenya’s economic indicators, digital infrastructure, and inequality trends using publicly available data.

The analysis focuses on understanding trends in economic growth, internet usage, mobile adoption, and social inequality.

Project Structure
World Development Indicators
→ GDP, population, and economic trends

Mobile Subscription
→ Growth of mobile adoption

Internet Users in Kenya
→ Internet usage trends

Inequality Trend
→ Income inequality patterns

Key Observations
Kenya shows steady growth in economic indicators over time
Mobile and internet usage have increased significantly
Inequality remains present across the observed period
Tools Used
Python (Pandas, NumPy, Matplotlib)
Jupyter Notebook
